🔧 How to Run PHP Scripts

Option 1: Find your PHP installation

Since you have Composer working, PHP is installed. Try these locations:

  • C:\xampp\php\php.exe
  • C:\wamp64\bin\php\php8.x.x\php.exe
  • C:\Program Files\PHP\php.exe

Option 2: Add PHP to PATH

Add your PHP directory to Windows PATH environment variable

Option 3: Use full path

Example: C:\xampp\php\php.exe preview_single_location.php
